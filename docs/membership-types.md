# Grey Sky Responder Portal - Membership Types

**Version:** 1.0 (Draft)  
**Date:** 2026-04-29  
**Author:** ATLAS (on Roy's confirmation)  
**Purpose:** Define membership structures to drive revenue, community, and alignment with FEMA RTLT standards. Prioritizes individuals while enabling organizational scaling. All types emphasize privacy: individuals own profiles; orgs get consent-based visibility.

## Core Principles
- **Revenue Focus:** Tiered pricing for sustainability ($100–$10,000+/yr based on scale).
- **Mission Alignment:** Tied to RTLT for credibility; fosters EM professional development.
- **Privacy-First:** No cross-org data sharing without explicit consent.
- **Flexibility:** Start simple, iterate based on beta feedback.
- **Portal Integration:** UI wireframe includes self-signup, dashboards, and admin approval for orgs.

## Membership Types

### 1. Individual Responder (Primary/Core)
- **Description:** Self-directed members pursuing EM certifications, training, or community access. Owns profile and data privacy.
- **Eligibility:** Open to anyone (responders, volunteers, aspiring professionals).
- **Features:** Profile storage, cert pathways (FEMA NQS/RTLT), forum access, training resources, badge issuance.
- **Pricing:** $100/yr (or $10/mo).
- **Revenue Model:** Volume-driven; upsell cert fees ($50–$200 per credential).
- **Portal Flow:** Instant signup, self-managed profile.

### 2. RTLT Team Organization (Priority Secondary)
- **Description:** Entities with teams configured/cited in FEMA RTLT (e.g., fire depts, SAR groups, hazmat teams).
- **Eligibility:** Any org with typed resources or teams pursuing RTLT alignment.
- **Features:** Team dashboards, bulk credentialing, readiness tracking, compliance reports. Scoped to sponsored members only.
- **Pricing:** $500–$2,000/yr (tiered by team size: <10, 10–50, 50+).
- **Revenue Model:** Base fee + per-member add-ons ($20/head for advanced tracking).
- **Portal Flow:** Admin approval; org admin invites members.

### 3. Government Entity (Governance-Focused)
- **Description:** Public sector orgs with operational EM roles, sub-tiered by level.
- **Eligibility:** Local (counties/cities), state (constitutional EM authority), federal (executive agencies like FEMA/USACE, legislative ops like AOC, judicial with ops factors like court marshals).
- **Features:** Multi-jurisdiction credentialing, integration with state/fed systems (e.g., EMAC), compliance reporting.
- **Pricing:** Local: $1,000/yr; State: $5,000/yr; Federal: $10,000+/yr (custom quotes).
- **Revenue Model:** Enterprise licensing + consulting upsells for integration.
- **Portal Flow:** Verified signup (e.g., gov email); hierarchical access (e.g., state oversees locals).

### 4. Private/Non-Profit Entity (Business & NGO)
- **Description:** Commercial and charitable orgs in disaster response.
- **Eligibility:** For-profits (e.g., recovery firms), non-profits/NGOs (e.g., Red Cross), tribes (sovereignty-respecting).
- **Features:** Workforce qualification tools, deployment tracking, custom standards alongside RTLT.
- **Pricing:** For-Profit: $2,000/yr; Non-Profit/NGO: $1,000/yr; Tribal: $1,000/yr.
- **Revenue Model:** Base + premium features (e.g., API access for $500/yr).
- **Portal Flow:** Application-based; tribal sub-tier includes custom fields for sovereignty.

### 5. Associate/Affiliate Group (Catch-All for Edges)
- **Description:** Informal or supporter groups without full RTLT teams (e.g., families sponsoring a responder, civic clubs like CERT auxiliaries).
- **Eligibility:** Families, local civic groups, or collectives with tangential EM ties. Not for unrelated entities.
- **Features:** Supporter roles—sponsor individuals, access basic training, community forums. Limited RTLT access; encourages upgrading to Team Org.
- **Pricing:** $250/yr (flat).
- **Revenue Model:** Low-volume entry point; upsell to higher tiers as they formalize.
- **Portal Flow:** Self-signup with light review; monitor for abuse/dilution.

## Perspectives & Tradeoffs
- **On Families/Civic Groups:** Yes to inclusion as affiliates if tied to responder support (e.g., family funding certs, civic group as volunteer base). Avoids dilution by limiting features—no team dashboards unless they qualify as RTLT. Beta test to gauge demand; could spin out if popular.
- **Missing Entities?** Covered: Governments (all branches/levels), companies (for/non-profit), tribes, teams, individuals, edges. Excludes unrelated (e.g., hobby groups)—redirect to individual if needed.
- **Risks:** Over-inclusion dilutes professional focus; mitigate with strict eligibility and upgrades.
- **Revenue Projection:** 70% from individuals/teams; 30% from gov/private. Aim for 1,000 users Year 1.
